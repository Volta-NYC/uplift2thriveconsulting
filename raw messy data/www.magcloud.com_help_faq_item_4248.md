## What is the maximum file size my PDF can be?

You can upload a PDF up to 300MB in size if you have the Adobe Flash Player or Microsoft Silverlight installed. Without one of these plug-ins installed in your browser, the maximum file size you can upload is 200MB.

You can find tips on how to reduce the file size of your PDF [here](https://www.magcloud.com/help/faq/item/6930).

[Click here](http://get.adobe.com/flashplayer/) to download the Adobe Flash Player, or [Click here](http://www.microsoft.com/silverlight/) to download Microsoft Silverlight.
