The Notifiable Data Breach (NDB) statistics dashboard presents key statistics on data breach notifications received by the OAIC under the [NDB scheme](https://www.oaic.gov.au/privacy/notifiable-data-breaches "https://www.oaic.gov.au/privacy/notifiable-data-breaches") since the scheme’s commencement in February 2018. The dashboard provides an interactive presentation of this data and will be updated twice-yearly. It has been created to help reporting entities and stakeholders understand the volume of data breaches reported to the OAIC, the number of people affected, causes and sectoral trends.

Data used in this dashboard have been produced and processed from information self-reported by entities under the [NDB scheme](https://www.oaic.gov.au/privacy/notifiable-data-breaches/about-the-notifiable-data-breaches-scheme).

For more details about the data used in this dashboard, refer to the **Data notes and disclaimer** page of the dashboard and [Notifiable data breaches](https://www.oaic.gov.au/privacy/notifiable-data-breaches).

## Using the dashboard

To use the dashboard:

- Apply filters to change the timeframe, or to filter by variables such as source of breach, response time or top industry sector.
- A button is included at the top left of all pages to provide further information about the data on the page, including additional instructions for how to the use the page.
- Right-click on a chart to view the chart data as a table.

We would love to learn more about your experience using the dashboard. You may leave us feedback by providing a rating under ‘Did you find this helpful?’

Information:

We are working on a mobile-friendly version of the dashboard. In the meantime, we recommend using a screen with a minimum display size of 1080 x 768 pixels for the best viewing experience.
