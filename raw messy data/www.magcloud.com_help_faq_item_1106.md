## How much does printing cost?

Printing costs vary based on the page size, product type and binding selection:

|     |     |     |     |     |
| --- | --- | --- | --- | --- |
| **Product Size** | **Price per Page\*** | **Half-Fold or Saddle Stitch** | **Perfect Binding** | **Wire-O Binding** |
| 5.25" x 8.25" | $0.17 | $0.00 | $1.00 | \-\-\- |
| 8.25" x 5.25" | $0.17 | $0.00 | $1.00 | \-\-\- |
| 8" x 8" | $0.20 | $0.00 | $1.00 | \-\-\- |
| 8.25” x 10.75” | $0.20 | $0.00 | $1.00 | \-\-\- |
| 10.75” x 8.25” | $0.20 | \-\-\- | \-\-\- | $2.00 |
| 12” x 12” | $0.36 | \-\-\- | \-\-\- | $2.00 |
| 11” x 14” | $0.36 | \-\-\- | \-\-\- | $2.00 |
| 14” x 11” | $0.36 | \-\-\- | \-\-\- | $2.00 |
| 17” x 11” | $0.36 | \-\-\- | \-\-\- | $2.00 |

_\*a page in this case is what you see numbered in a book or magazine: one side of one physical page in a printed publication_

5.25" x 8.25" and 8.25" x 5.25" Flyers cost $0.36 per piece for printing and [UV coating](https://www.magcloud.com/help/faq/item/4120) is available for an additional $0.06 per piece.

8" x 8", 8.25" x 10.75" and 10.75" x 8.25" Flyers cost $0.40 per piece for printing and [UV coating](https://www.magcloud.com/help/faq/item/4120) is available for an additional $0.12 per piece.

Posters cost $2.00 per piece for printing and include [UV coating](https://www.magcloud.com/help/faq/item/4120) on both sides.

All orders of 10 or more print copies receive a discount off the printing costs.

|     |     |     |     |
| --- | --- | --- | --- |
| **1 - 9 copies** | **10 - 49 copies** | **50 - 99 copies** | **100+ copies** |
| Base price | 15% | 20% | 25% |

[Click here](https://www.magcloud.com/products/pricing) to use our cost calculator to see the expected cost for your own order and your customers' orders.
